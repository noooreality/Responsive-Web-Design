Einführung

JavaScript ist eine weit verbreitete Programmiersprache für das Web.

Sie wird hauptsächlich im Frontend eingesetzt.

console.log("Hallo Welt");

JavaScript kann direkt im Browser ausgeführt werden.

Es wurde 1995 entwickelt und ständig weiterentwickelt.

JS ist eine interpretierte Sprache.

Sie wird oft zusammen mit HTML und CSS verwendet.

Sie ist sehr flexibel und dynamisch.

Mit JavaScript lassen sich interaktive Webseiten erstellen.

  • Interaktivität
  • Browserkompatibilität
  • Große Community
Variablen und Typen

Es gibt drei Hauptarten: var, let und const.

let und const sind ab ES6 empfohlen.

let zahl = 5; const name = "Max";

JavaScript ist dynamisch typisiert.

Datentypen umfassen Number, String, Boolean, Object, etc.

Typeof kann zur Typüberprüfung verwendet werden.

Null und undefined sind unterschiedliche Werte.

Primitive Typen sind unveränderlich.

  • Number
  • String
  • Boolean
Funktionen

Funktionen sind grundlegende Bestandteile in JS.

Sie werden mit dem Schlüsselwort function deklariert.

function hallo() { console.log("Hallo!"); }

Arrow Functions wurden mit ES6 eingeführt.

const add = (a, b) => a + b;

Funktionen können Parameter und Rückgabewerte haben.

Funktionen sind Objekte.

Sie können auch anonym sein.

Kontrollstrukturen

JS unterstützt if, else, switch, while, do-while und for.

Bedingungen basieren auf Boolean-Werten.

if (x > 10) { console.log("Größer als 10"); }

Auch ternäre Operatoren sind möglich.

switch ist nützlich für viele Bedingungen.

while-Schleifen wiederholen sich, solange die Bedingung wahr ist.

for (let i = 0; i < 5; i++) { console.log(i); }
DOM und Events

Der DOM stellt die HTML-Struktur als Objekt dar.

Mit JS kann man HTML-Elemente manipulieren.

document.getElementById("demo").innerText = "Hallo";

Event Listener reagieren auf Benutzeraktionen.

button.addEventListener("click", funktion);

Events umfassen click, submit, keydown usw.

  • change
  • input
